name: datamanager-inspector
description: See how the freecad datamanager workbench implements something
---

The datamanager project is stored in the same location as this project. Just go up one directory from the root project, and cd into `freecad_datamanager_workbench`, e.g. `cd ../freecad_datamanager_workbench`. You may grep and search through files to see how it implements any functionality.
