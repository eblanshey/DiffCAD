The freecad source repository is stored in the same location as this project. Just go up one directory from the root project, and cd into `freecad-source`, e.g. `cd ../freecad-source`. You may grep and search through files to see how it implements any functionality. You may instruct the speedy "explore" subagent to find relevant files.in
